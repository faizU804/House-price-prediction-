## 1. Project Overview

### Concept
A high-performance machine learning application that predicts residential real estate prices in Melbourne, Australia. It bridges the gap between complex data science and user-friendly web interfaces.

### Technology Stack
- **Backend**: Python 3.14+ (Flask Framework for the web layer).
- **Machine Learning**: Scikit-Learn (using Gradient Boosting Regressor).
- **Frontend**: HTML5, CSS3 (Vanilla + Bootstrap 5), AOS (Animate on Scroll).
- **Deployment**: Vercel (Serverless Python).

### How it Works
1. **Data**: Uses the Melbourne Housing Dataset (`melb_data.csv`).
2. **Preprocessing**: Automatically handles missing values and encodes categorical data.
3. **Model**: A Gradient Boosting model retrained for maximum compatibility and accuracy (R² ~ 0.825).
4. **Interface**: Users enter property details (Rooms, Region, Type, etc.), and the system provides an instant price estimate.
